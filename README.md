# ui-components-mcp

## 安装依赖

```sh
cp .env.sample .env
yarn
```

## 修改配置

下载代码仓库：

```sh
git clone https://github.com/vibeus/design-system.git
git clone https://github.com/radix-ui/website.git
```

修改 app/comp-mcp-server/lib/paths.ts 下的相关路径

## 运行

```sh
yarn dev
```
